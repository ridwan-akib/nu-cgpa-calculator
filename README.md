 NU CGPA Calculator

এটি একটি সহজ এবং কার্যকর জিপিএ ক্যালকুলেটর যা বিশেষভাবে জাতীয় বিশ্ববিদ্যালয়ের (National University) শিক্ষার্থীদের জন্য তৈরি। এর মাধ্যমে আপনি সহজেই আপনার সেমিস্টার বা ইয়ারের রেজাল্ট ক্যালকুলেট করতে পারবেন।

 ফিচারসমূহ (Features)
- **সহজ ইন্টারফেস:** দ্রুত এবং নির্ভুল হিসাব করার সুবিধা।
- **ক্রেডিট বেসড ক্যালকুলেশন:** জাতীয় বিশ্ববিদ্যালয়ের গ্রেডিং সিস্টেম অনুযায়ী ক্রেডিট আওয়ার ও জিপিএ হিসাব।
- **মোবাইল ফ্রেন্ডলি:** যেকোনো স্মার্টফোন বা কম্পিউটার থেকে ব্যবহারযোগ্য।

 কীভাবে ব্যবহার করবেন (How to use)
১. ওয়েবসাইটটি ভিজিট করুন: https://ridwan-akib.github.io/nu-cgpa-calculator/

২. প্রতিটি কোর্সের জন্য ক্রেডিট আওয়ার এবং প্রাপ্ত লেটার গ্রেড বা পয়েন্ট ইনপুট দিন।
৩. 'Calculate' বাটনে ক্লিক করুন।
৪. আপনার সিজিপিএ বা জিপিএ রেজাল্টটি সাথে সাথে দেখতে পাবেন।
